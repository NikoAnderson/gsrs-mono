G-SRS v3.1 (December 14, 2023)


CHANGELOG

GSRS 3.1 software release is a major release incorporating new features in data exchange, which introduces staging area for importing, user saved lists, and structure rending Improvements in substances module. A document containing an in-depth description of the highlighted features and bug fixes / improvements and more can be downloaded Here New Feature And Enhancement Highlights

Full Release Highlights / Notes
  • Data Exchange - file import (ETL Framework)
  • Bulk User saved lists
  • New features and Improvements in Product, Application, Clinical Trials, Impurities modules
  • structure rendering improvements
  • Quick Registration Tools
  • Developer SOP for database schema changes
  • ASCII / markup name support
  • Notes and validation message rework and custom styling
  • Impurities and produckt module improvements
  • Updated, refined, and refactored definitional hash algorithm
  • Structure search quality of life improvements
